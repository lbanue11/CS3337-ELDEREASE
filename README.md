# Elder Ease

Elder Ease is a web application designed to help elderly users in Los Angeles find local resources such as pharmacies and care facilities. The project also supports caregiver features, including account management and communication tools. I worked mainly on the backend using Java, Spring Boot, and PostgreSQL. My work included building features for registration, login, user profiles, favorites, reviews, and caregiver-elder interactions. This project gave me experience with backend development, relational databases, and team-based software development.
